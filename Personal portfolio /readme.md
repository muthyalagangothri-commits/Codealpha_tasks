Personal Portfolio Website

A modern and responsive full-stack portfolio website built using HTML, CSS, JavaScript, React.js, Node.js, Express.js, and MongoDB. The website showcases my profile, skills, projects, education, and contact information. It features a clean user interface, responsive design, and a contact form for visitors. This project demonstrates full-stack development skills, including frontend design, backend API development, and database integration.
